При запуске программа вначале запрашивает имя входного файл
Файл должен иметь расширение *.xlsx или *.ods
После ввода имени файла программа запрашивает имя страницы в этом файле(default: Var) с входными данными
Пример входного файла: mv.xlsx
Пример входной страницы в входном файле: Var
Ответ выводиться в файл с названием 'Имя входного файла'_ans.xlsx
